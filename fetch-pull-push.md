# git fetch, git pull y git push

Estos tres comandos sincronizan el repositorio local con el repositorio remoto.

## git fetch

Descarga los cambios del repositorio remoto sin aplicarlos en los archivos de trabajo.

**Ejemplo:** 
git fetch origin

## git pull

Descarga los cambios del repositorio remoto y los aplica a los archivos de trabajo.

**Ejemplo:** 
git pull origin main

## git push

Sube los commits realizados en el repositorio local al repositorio remoto.

**Ejemplo:** 
git push origin alumno4_rama
