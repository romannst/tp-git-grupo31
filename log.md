# git log:
Muestra un historial de todos los commits del repositorio en orden cronológico, detallando el hash que le corresponde a cada commit, el autor, la fecha en la que se realizo y el mensaje.
# git log --oneline:
Muestra todos los commits pero resumido en una sola línea, mostrando solo el principio del hash y su mensaje.
# git log --graph --oneline:
Al comando anterior le agrega un gráfico para visualizar las distintas ramas y merges correspondientes a cada commit.
# git log -p:
Muestra la cantidad de modificaciones de archivos entre commits.
